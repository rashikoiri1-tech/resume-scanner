# 🌟 Overview

ResumeScanner AI is an intelligent web application that analyzes a user's resume and matches it with thousands of company job requirements.

The platform identifies:

✅ Missing Skills

✅ Resume Strength Score

✅ ATS Compatibility Score

✅ Suitable Job Roles

✅ Recommended Companies

✅ Skill Gap Analysis

✅ Personalized Learning Roadmap

✅ AI Career Suggestions

The system helps students, freshers, and professionals improve their resumes and increase their chances of getting shortlisted.

---

# 🎯 Problem Statement

Many candidates apply to jobs without knowing:

- Whether their resume matches company requirements
- Which skills are missing
- Which companies they are eligible for
- Their ATS score
- How to improve their profile

As a result:

❌ Resume rejection rate increases

❌ Candidates waste time applying to irrelevant jobs

❌ Skill gaps remain unidentified

❌ Placement opportunities are missed

---

# 💡 Solution

ResumeScanner AI solves this problem by:

1. Uploading Resume (PDF/DOCX)
2. Extracting Skills Automatically
3. Comparing Skills with Company Requirements
4. Calculating ATS Compatibility Score
5. Identifying Missing Skills
6. Recommending Matching Companies
7. Suggesting Courses to Learn Missing Skills
8. Generating Personalized Career Insights

# 🚀 Installation Guide

## Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/resume-scanner-ai.git
```

## Step 2: Enter Project Folder

```bash
cd resume-scanner-ai
```

---

## Step 3: Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## Step 4: Install Backend Dependencies

```bash
cd ../backend
npm install
```

---

## Step 5: Install Python Dependencies

```bash
pip install -r requirements.txt
```

---

## Step 6: Configure Environment Variables

Create `.env`

```env
PORT=5000

MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret

OPENAI_API_KEY=your_api_key
```

---

## Step 7: Start Backend

```bash
npm run server
```

---

## Step 8: Start Frontend

```bash
npm start
```
